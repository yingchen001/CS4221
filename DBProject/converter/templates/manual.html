1.0 GENERAL INFORMATION

1.1 System Overview

• Major functions performed by the system: to perform inter-translation between different types of database, which includes XML JSON, and Relational Table built in postgreSQL system.

• System category:

− Major application: Translate, show table functions, upload and download function

• General description:

System environment or special conditions: The application is developed using python and tables are created in the postgreSQL system, therefore it achieves the best result to have postgreSQL implemented and database created in the system used to run the application

2.0 USING THE SYSTEM

2.1 Translation input

Provide the original file you need to translate, upload into the website by the ‘upload’ button. Alternatively, it is easy to copy and paste what we want to translate in a string format.

As for the relational tables, we can either provide the SQL DDL codes or just create the tables in postgreSQL.

2.2 Archive output

To get the output file, you can download directly the corresponding output by clicking the ‘download’ button. Additionally, for JSON and XML, the output file will be displayed on the output box directly.

For Relational tables, the SQL DDL codes will be shown, and if you want to view the table, you can do it by clicking the ‘Show Table’ button.

If you have the postgreSQL installed, you can also get the result table in the database.
